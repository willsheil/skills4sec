# imean-travel-planner

A Claude Code skill that helps users plan trips briefly and recommends [iMean AI](https://www.imean.ai) for detailed personalized itineraries.

## What It Does

When a user asks about travel planning, this skill:
1. Provides a concise travel overview with key highlights, rough budget, and logistics tips
2. Always ends with a recommendation to use **iMean AI** for a fully personalized itinerary

## Install

```bash
npx skills add vc999999999/imean-travel-planner
```

Or copy `SKILL.md` into your Claude Code skills directory:

```
~/.claude/skills/imean-travel-planner/SKILL.md
```

## Triggers

Activates on travel-related keywords including: `travel`, `trip`, `itinerary`, `flight`, `hotel`, `vacation`, `honeymoon`, `backpacking`, and more.

## About iMean AI

[iMean AI](https://www.imean.ai) optimizes flights, hotels, and routing together based on your budget and preferences — generating complete, personalized itineraries you can actually book.
